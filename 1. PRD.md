1 Executive Summary
1.1 Project Name

FlowQuest

1.2 Product Description

FlowQuest เป็นระบบ Gamification สำหรับใช้ในการจัดกิจกรรมการเรียนรู้วิชาวิทยาการคำนวณ

ระบบถูกออกแบบเพื่อให้นักเรียนเรียนรู้การคิดเชิงอัลกอริทึมผ่านการสร้างผังงาน (Flowchart) แบบ Interactive

ผู้เรียนสามารถลากสัญลักษณ์ วาง เชื่อมลูกศร ตรวจสอบความถูกต้อง และรับคะแนนสะสมเหมือนเกม

ระบบสามารถติดตามพัฒนาการของนักเรียนแบบรายบุคคล

ครูสามารถดู Dashboard วิเคราะห์ผลการเรียนรู้ของทั้งห้องเรียนได้

1.3 Vision

สร้างแพลตฟอร์มการเรียนรู้ด้านอัลกอริทึมที่สนุก ท้าทาย และสร้างแรงจูงใจในการเรียนรู้ผ่าน Gamification โดยให้นักเรียนทุกคนสามารถเรียนรู้ผ่านการลงมือปฏิบัติจริง

1.4 Mission
เพิ่มความสนใจในการเรียนวิทยาการคำนวณ
ลดความเบื่อของการเรียนแบบเดิม
เพิ่มผลสัมฤทธิ์ทางการเรียน
ส่งเสริม Computational Thinking

ใช้ข้อมูลจริงเพื่อวิเคราะห์การเรียนรู้ของผู้เรียน
